# 시스템정보 명령어

## ps
- process status 출력

### 주로 쓰는 옵션
- -e : 모든 프로세스
- -f : Full-format
- -L : 스레드까지 출력 
- --forest : 부모자식간의 관계를 트리형태로 구성하여 출력
- -o : 출력 포맷
- --sort : 

```bash
$ ps
$ ps -ef
$ ps -ef | grep 8080
$ ps --ppid 1515 -f
$ ps -ef --forest

$ ps -eF --sort=%mem | head     # 오름차순
$ ps -eF --sort=-%mem | head    # 내림차순
$ ps -eF --sort=ppid
$ ps -eF --sort=ppid,rss

$ ps -eo pid,ppid,args,c --sort pid
```
